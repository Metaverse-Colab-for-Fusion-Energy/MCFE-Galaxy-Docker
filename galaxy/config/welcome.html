Hello RSEs! Welcome to my show and tell

This Galaxy instance has a few example tools and workflows to help you get started.

In Tools -> Example Tools you will find:

  • Word Count: a tool that counts the number of words in a text file.
  • Letter Count: a tool that counts the number of letters in each word in a text file.
  • Histogram: a tool that generates a histogram of the letter counts

In Workflows -> Public Workflows you will find:

  • Example tools workflow: a simple workflow that uses the example tools above.

Feel free to run and play with these tools and workflows!

For more information on how to use Galaxy, please visit the Galaxy Training Network.

A guide to configuring this demo, adding tools and workflows, and customising the interface can be found in the GitHub repository for this demo.

To raise issues or ask questions about this Galaxy instance, please visit the GitHub issues page.
